Romania Public Transport API

Examples

  • Cities - GET /v0.1/cities
  • Brasov
    • Capabilities GET /v0.1/capabilities
    • Agency Info GET /v0.1/{string:city}/agency
    • Stops - GET /v0.1/{string:city_id}/stops
    • Routes - GET /v0.1/{string:city_id}/routes
    • Route from stop A to stop B - GET /v0.1/route?start_stop_id={string:start_stop_id}&end_stop_id={string:end_stop_id}
  • Bucuresti
    • Stops - GET /v0.1/{string:city_id}/stops
    • Routes - GET /v0.1/{string:city_id}/routes
    • Route from stop A to stop B - GET /v0.1/route?start_stop_id={string:start_stop_id}&end_stop_id={string:end_stop_id}
Protocols:
  • GTFS and GTFS Realtime
  • JSON
  • XML
  • CSV
  • Protocol buffers/gRPC
